How it works

From connection to insights in five minutes.

  1. Connect your Google Search Console property — we pull verified metrics securely.
  2. Add a Page — paste the canonical URL you want to optimize.
  3. Create Snippet Variants (A/B/C) — titles & meta descriptions.
  4. Schedule Variant Runs — set start dates and durations (no overlaps).
  5. Track Daily Results — impressions, clicks, CTR, position per run.

Do I need to install anything on my site?

No. We read performance data from Google Search Console and you update titles/descriptions in your CMS as you rotate variants.

How do runs avoid overlapping?

Our scheduler enforces non-overlapping planned/actual windows per test. If dates collide, you’ll be asked to adjust before saving.

Which metrics are tracked?

Daily impressions, clicks, CTR and average position by variant run window—exactly as reported by GSC.

What’s a good default duration?

We default to 14 days per variant, but you can set any duration per test or per variant depending on traffic.

Can I export results?

Yes—CSV export is available on Pro and Agency plans for client reporting or deeper analysis.
